  • 8年专业前端开发，13年全流程项目落地经验。擅长复杂业务系统与软硬件一体化产品的前端全链路开发。
  • 具备从0到1的前端架构设计、技术选型、性能优化及全周期项目交付能力。
  • 熟悉Vue2/Vue3、UniApp跨端开发框架核心技术栈，熟练运用 TypeScript/React 进行业务功能开发。
  • 掌握前端工程化、WebRTC音视频、多厂商硬件SDK集成、国产化适配、老旧系统重构与性能优化等核心技能。
  • 担任核心产品线前端技术负责人，主导智能终端应用、远程会见平台、智能视频分析系统等多款产品迭代，覆盖公安司法监所、航空、文旅电商等领域。
  • 具有多个大型项目全流程前端交付经验，包括招投标支持、开发测试、现场部署与项目验收。
  • 拥有5年工业级监控系统组态设计与工程实施背景，具备扎实的工程化思维、深入的业务理解能力，能够高效推进跨团队协作与项目落地。

工作经历

广州市高科通信技术股份有限公司 | 开发工程师 | 2020.06 - 2026.02

  • 角色与成果：作为公司核心产品线的前端技术负责人，具备6年一线开发与项目管理经验。主导了从技术架构设计、核心功能开发、多端适配到全国项目落地支持的全流程，深度参与超十个大型项目的招投标支持与最终验收。
  • 核心项目一（智能终端产品线）：
    • 架构与开发：主导“智能终端”系列（V1-V4）前端架构设计，并独立完成仓内屏/仓外屏/管控主机等App全部功能开发，实现多厂商硬件适配。
    • 主导开发：主导平板端独立应用的主体开发（完成度>80%），其出色的可扩展性与用户体验使其成为公司拳头产品，获客户高度评价。
  • 核心项目二（司法协作平台）：
    • 全面接管与重构：在同事离职后，全面接手“律师/家属会见与远程提讯平台”，主导其后续的架构调整、性能优化与功能迭代（完成度>80%）。
    • 平台整合：推动其整合为统一的“三合一”平台，客户满意率达95%以上。
  • 核心项目三（智能分析平台）：
    • 迭代与国产化：承接由中山大学团队交付的第一版“智能视频行为分析平台”，独立负责所有后续的UI整体改版与功能迭代。
    • 开发基础版与国产化适配版，国产化版本已成为现场部署的主流版本。

广州市唯景互娱信息技术有限公司 | 前端开发工程师 | 2018.10 - 2020.06

  • 核心交付：负责多个交付项目的前端开发，独立承担复杂业务实现与技术方案落地。接手“飞行模拟机排班系统”项目后，协同完成项目交付，快速响应解决现场问题保障项目成功验收，获得客户书面表彰、优秀员工和涨薪奖励。
  • 技术主导：主导“西域游”电商平台及“训练资料电子化系统”的前端技术选型、框架搭建与核心模块开发。
  • 跨端实践：针对“训练资料电子化系统”的平板端需求，选用UniApp框架，独立完成项目前期搭建与核心功能开发。

深圳市共济科技股份有限公司 | 组态设计工程师 | 2013.07 - 2018.06

  • 专业交付：基于自研CM-Desk平台，负责数据中心监控系统的组态设计与部署，实现电力/空调/安防等设施的集中监控。累计参与项目超300个，并独立完成数百个小型机房的组态设计与部署，平均每月交付20+项目。
  • 项目管理：担任项目组长及部门经理助理，协调项目资源、推进进度，并负责新员工的技术培训与指导。

项目经历

智能终端系列应用与实战平台（V1 - V4） | 2020.06 - 2026.02

  • 项目描述：面向监所行业的软硬件一体化解决方案，涵盖定制安卓终端（仓内屏、仓外屏、管控主机）、独立（平板、手机）应用及Web端实战平台。产品已成功部署于广东省内十几个监所及省外项目。
  • 职责与贡献：
    • 主导终端应用全栈开发与架构演进：独立负责终端App（V1-V3）从0到1的前端架构设计与功能实现（UniApp/Vue2）。在V4版本主导技术栈升级至Vue3/HBuilder4.5，并完成UI整体改版和功能开发。
    • 独立开发平板端核心产品：作为主程，独立开发了集成谈话教育、智慧医疗等十余项核心功能的平板端App，支持离线/在线一体化操作，个人完成度超80%。
    • 负责后台系统重构与核心模块开发：独立维护并迭代V1-V3旧版后台管理系统（Vue2/iView）。在V4实战平台升级中，独立重写了核心的“监室事务管理”主模块（含十余个子模块），基于Vue3/TS/Vite/Element Plus技术栈实现。
    • 负责硬件集成与工程落地：负责集成并调试大华、海康、来邦等多家硬件设备的SDK，解决兼容性问题。使用Android Studio完成开发基座打包与正式APK生成。

律师/家属会见与远程提讯三合一平台 | 2022.03 - 2024.10

  • 项目描述：整合律师会见、家属会见与远程提讯功能的Web实时音视频协作平台，大幅提升系统稳定性和交付标准化，客户满意度达95%以上。
  • 职责与贡献：
    • 全面接手并主导迭代：在前任开发离职后，全面接手项目，主导后续的架构调整、性能优化与功能迭代，个人完成后续开发工作的80%以上。
    • 推动平台整合与产品化：主导将原先独立的三个系统整合为统一的“三合一”平台，实现了代码复用与统一维护，使其成为公司标准交付件。
    • 维护核心通信与硬件集成：负责WebRTC音视频通信、Socket.IO信令服务等核心模块的维护与优化，并集成高拍仪、电子签名板等外设。

智能视频行为分析平台（V1 - V4） | 2023.03 - 2024.08

  • 项目描述：用于视频监控智能分析的Web平台，支持多路视频流接入与多种AI行为分析算法。已稳定部署于广州白云、中山、花都、江门等多个监所。平台算法识别率稳定在90%以上，国产化版本成为主流部署方案。
  • 职责与贡献：
    • 独立承接并负责全部前端开发：在中大团队交付第一版后，独立接手项目所有前端工作，主导完成平台的UI整体改版、布局优化及功能迭代。
    • 实现多版本并行开发与适配：根据部署需求，同时开发并维护“基础版”与“国产化适配版”两个并行版本，保障了项目平稳过渡。

珠海翔翼飞行模拟机排班系统 | 2018.10 - 2020.06

  • 项目描述：珠海翔翼公司的内部模拟机排班运营系统。接手离职同事项目，协同另一前端同事完成剩余80%+功能开发。快速响应客户需求解决问题，有力推动项目终验，获客户书面表彰及公司年度优秀员工、涨薪奖励。
  • 职责与贡献：
    • 负责核心模块开发与联调：独立负责排班管理、通知中心、客户信息管理及数据分析看板等核心功能的前端开发与接口联调。
    • 实现实时通信与工程支持：封装全局网络请求工具，基于WebSocket实现实时消息推送与数据更新。参与系统测试、上线与问题排查，保障项目顺利交付。

其他项目经历

  • 西域游-新疆旅游电商平台​ (2018.10-2020.06)：负责技术选型与Vue项目框架搭建，独立完成用户端首页、登录注册、商品列表/详情、搜索等核心页面开发与组件封装。远程协同合作公司前端工程师完成项目整体开发和交付。
  • 训练资料电子化系统​ (2019.10-2020.06)：独立完成基于uni-app的配套平板APP开发，实现飞行员训练资料的快速查阅。

早期经历（数据中心监控管理系统​） (2013.07-2018.06)

  • 核心职责：负责CM-Desk组态监控平台的设计与策略配置，实现对精密配电、空调、UPS等设备的实时监控与智能告警。
  • 代表性项目：参与平安国际金融中心/太平金融大厦数据中心、深圳证券交易所运营中心、深圳海关/兴业银行多地数据中心等多个高可靠性项目的监控系统实施。
  • 团队贡献：主导建立部门组态模板库，将经验标准化，显著提升团队开发效率和缩短项目实施周期。兼任部门经理助理，协助项目管理与协调。
  • 成果：兼任部门经理助理，协助项目管理与协调。因表现突出获评公司优秀员工，参与项目获评年度优秀项目。

专业技能

核心前端技术与框架

  • 前端架构与全栈能力：熟悉 Vue.js 2/3 及其生态 (Vue Router, Vuex)，具备独立设计前端架构、主导大型项目（如智能终端平台、视频分析平台）技术升级与重构（Vue2 至 Vue3）的能力。能够使用 React 技术栈进行功能开发与界面迭代。
  • 工程化与开发规范：熟练掌握 Vite、Webpack 等构建工具，具备基于 Vue3/TypeScript/Vite 等现代技术栈从零搭建并优化大型工程的能力。熟悉 ESLint 代码规范、模块化开发及 Docker 容器化部署。

跨端移动开发与原生能力

  • 跨端应用开发：精通 UniApp 框架，拥有多款终端 App（Android）的全量独立开发与主导升级经验。熟悉 NVue 原生渲染、HTML5+ Runtime 调用设备原生能力。
  • 原生集成与发布：熟练使用 Android Studio 进行调试、制作开发基座与正式 APK 打包发布。具备集成 SQLite、SIP/对讲协议等原生模块以及多家硬件厂商 SDK 的实战经验。

实时通信、音视频与硬件集成

  • 实时通信与音视频：熟悉 WebSocket/Socket.IO 实时通信及 WebRTC 音视频技术，拥有构建与维护高稳定性实时音视频协作平台（如律师会见三合一平台）的完整经验。
  • 硬件与第三方集成：熟练掌握与海康/大华/来邦/昆仑等多家安防及通讯硬件厂商 API/SDK 的集成、联调与问题排查。具备集成流媒体（HLS, video.js）、高拍仪、电子签名板等外设的经验。

工程实践与综合能力

  • 全流程项目经验：具备深入参与政企项目全流程的经验，包括技术方案设计、核心模块开发、多版本（基础版/国产化版）适配、远程支持部署及线上问题快速定位解决。
  • 复杂系统协作与优化：拥有接手并主导迭代复杂系统（如智能视频分析平台）、进行性能优化与架构调整的能力。具备优秀的跨职能（前端、硬件、后端）协同与复杂问题攻坚能力。
