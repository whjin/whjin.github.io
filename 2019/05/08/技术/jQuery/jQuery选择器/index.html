еҹәжң¬йҖүжӢ©еҷЁ Basics

  • *йҖүжӢ©жүҖжңүе…ғзҙ 
  • .classйҖүжӢ©classпјҢеҰӮпјҡ$('.mybox')
  • elementйҖүжӢ©elementпјҢеҰӮпјҡ$('p')
  • #idйҖүжӢ©idпјҢеҰӮпјҡ$('#box')
  • selector1,selectorNеҸҜд»ҘеҗҢж—¶йҖүжӢ©еӨҡдёӘе…ғзҙ пјҢеҰӮпјҡ$('div, p.box, #phone')

еұӮзә§йҖүжӢ©еҷЁ Hierarchy

  • parent > childйҖүжӢ©жҢҮе®ҡе…ғзҙ дёӢзҡ„жҢҮе®ҡеӯҗе…ғзҙ пјҢеҰӮпјҡ$('ul.tonav > li')
  • ancestor descendantйҖүжӢ©дёҖдёӘе…ғзҙ йҮҢжүҖжңүзҡ„еҗҺд»Је…ғзҙ пјҢеҰӮпјҡ$('form input')
  • prev + nextйҖүжӢ©жүҖжңүжҢҮе®ҡе…ғзҙ еҗҺзҙ§и·ҹзқҖзҡ„е…ғзҙ пјҢеҰӮпјҡ$('label + input')
  • prev ~ siblingsйҖүжӢ©дёҺжҢҮе®ҡе…ғзҙ д№ӢеҗҺжңүзӣёеҗҢзҲ¶зә§зҡ„еҗҢзә§йҖүжӢ©еҷЁпјҢеҰӮпјҡ$('#prev ~ div')

еҹәжң¬иҝҮж»ӨеҷЁ Basic Filters

  • :animatedйҖүжӢ©еҠЁз”»иҝӣиЎҢдёӯзҡ„жүҖжңүе…ғзҙ гҖӮеҰӮпјҡ$('div:animated)
  • :eq(n)йҖүеҸ–з¬¬nдёӘе…ғзҙ пјҢеҰӮпјҡ$('ul.tonav li:eq(n)')
  • :evenйҖүеҸ–еҒ¶ж•°дёӘе…ғзҙ пјҢеҰӮпјҡ$('li:even')
  • :oddйҖүеҸ–еҘҮж•°дёӘе…ғзҙ пјҢеҰӮпјҡ$('li.odd')
  • :firstйҖүеҸ–еҘҮж•°дёӘе…ғзҙ пјҢеҰӮпјҡ$('li:first')
  • :gt(n)йҖүеҸ–з»“жһңйӣҶдёӯзҙўеј•еӨ§дәҺnзҡ„е…ғзҙ пјҢnеҸҜд»ҘдёәиҙҹеҖјпјҢеҰӮпјҡ$(':gt(3)')
  • :lt(n)йҖүеҸ–з»“жһңйӣҶдёӯзҙўеј•е°ҸдәҺnзҡ„е…ғзҙ пјҢnеҸҜд»ҘдёәиҙҹеҖјпјҢеҰӮпјҡ$(':lt(3)')
  • :headerйҖүеҸ–жүҖжңүзҡ„ж Үйўҳе…ғзҙ пјҢдҫӢеҰӮh1гҖҒh2гҖҒh3вҖҰпјҢеҰӮпјҡ$(':header')
  • :lang()йҖүеҸ–жҢҮе®ҡиҜӯиЁҖзҡ„жүҖжңүе…ғзҙ пјҢпјҢеҰӮпјҡ$('div:lang(zh-cn)')
  • :lastйҖүеҸ–жңҖеҗҺдёҖдёӘз¬ҰеҗҲзҡ„е…ғзҙ пјҢеҰӮпјҡ$('li:last')
  • :notйҖүеҸ–дёҚз¬ҰеҗҲзҡ„жүҖжңүе…ғзҙ пјҢеҰӮпјҡ$('input:not(:checked) + span')
  • :rootйҖүеҸ–дҪңдёәж–ҮжЎЈж №зӣ®еҪ•зҡ„е…ғзҙ 
  • :targetйҖүеҸ–з”ұж–ҮжЎЈзҡ„еӣҫзүҮгҖҒи§Ҷйў‘гҖҒйҹійў‘жҢҮзӨәзҡ„зӣ®ж Үе…ғзҙ 

еҶ…е®№иҝҮж»ӨеҷЁ Content Filters

  • :contains()йҖүжӢ©еҢ…еҗ«жҢҮе®ҡж–Үжң¬зҡ„жүҖжңүе…ғзҙ пјҢеҰӮпјҡ$("div:containers('home')")
  • :emptyйҖүжӢ©жІЎжңүеӯҗе…ғзҙ жҲ–еҶ…е®№ж–Үеӯ—зҡ„е…ғзҙ пјҢеҰӮпјҡ$("td:empty"))
  • :has()йҖүжӢ©еҢ…еҗ«иҮіе°‘дёҖдёӘеҢ№й…ҚжҢҮе®ҡйҖүжӢ©еҷЁзҡ„е…ғзҙ зҡ„е…ғзҙ пјҢеҰӮпјҡ$("div:has(p)"))
  • :parentйҖүжӢ©иҮіе°‘жңүдёҖдёӘеӯҗиҠӮзӮ№пјҲе…ғзҙ жҲ–ж–Үжң¬пјүзҡ„жүҖжңүе…ғзҙ 

еҸҜи§ҶйҖүжӢ©еҷЁ Visibility Filters

  • :hiddenйҖүжӢ©жүҖжңүйҡҗи—Ҹзҡ„е…ғзҙ пјҢеҰӮпјҡ$("div:hidden").show(3000));
  • :visibleйҖүжӢ©жүҖжңүйҡҗи—Ҹзҡ„е…ғзҙ пјҢеҰӮпјҡ
    $("div:visible").click(function() {$(this).css("background", "yellow");});

еұһжҖ§йҖүжӢ©еҷЁ Attribute

  • [name]йҖүжӢ©е…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҲдҪҝз”Ёд»»дҪ•еҖјйғҪеҸҜд»ҘпјүгҖӮеҰӮпјҡ$("div[id]")

  • [name|="value"]йҖүжӢ©е…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҢе…¶еҖјзӯүдәҺз»ҷе®ҡзҡ„еӯ—з¬ҰдёІпјҢжҲ–иҖ…д»ҘиҜҘеӯ—з¬ҰдёІејҖеӨҙпјҢеҗҺи·ҹиҝһеӯ—з¬ҰпјҲ-пјүгҖӮ

  • [name*="value"]йҖүжӢ©е…·жңүеҢ…еҗ«з»ҷе®ҡеӯҗеӯ—з¬ҰдёІзҡ„еҖјеҫ—жҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ гҖӮ

  • [name~="value"]йҖүжӢ©е…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҢе…¶дёӯеҢ…еҗ«з”ұз©әж јеҲҶйҡ”зҡ„з»ҷе®ҡеҚ•иҜҚзҡ„еҖјгҖӮ

  • [name$="value"]йҖүжӢ©е…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҢе…¶еҖјдёҺз»ҷе®ҡеӯ—з¬ҰдёІзІҫзЎ®еҢ№й…ҚпјҢиҰҒеҢәеҲҶеӨ§е°ҸеҶҷгҖӮеҰӮпјҡ$("input[name$='letter']")

  • [name="value"]йҖүжӢ©е…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҢе…¶еҖјжҒ°еҘҪзӯүдәҺзү№е®ҡеҖјпјҢеҰӮпјҡ$("input[value='Hot Fuzz']")дјҡйҖүеҲ°valueзӯүдәҺHot Fuzzзҡ„input

  • [name!="value"]йҖүжӢ©дёҚе…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҢжҲ–иҖ…е…·жңүжҢҮе®ҡиӢҸйҶ’дҪҶдёҚе…·жңүзү№е®ҡеҖјзҡ„е…ғзҙ гҖӮеҰӮпјҡ$("input[name!='newsletter']")пјҢnameеұһжҖ§еҖјдёәnewsletterзҡ„дёҚйҖүгҖӮ

  • [name^="value"]йҖүжӢ©е…·жңүжҢҮе®ҡеұһжҖ§зҡ„е…ғзҙ пјҢе…¶еҖјдёҺз»ҷе®ҡеӯ—з¬ҰдёІе®Ңе…ЁдёҖиҮҙгҖӮ

  • [name="value"][name2="value2"]з¬ҰеҗҲжүҖжңүжҢҮе®ҡзҡ„еұһжҖ§иҝҮж»ӨеҷЁзҡ„е…ғзҙ гҖӮеҰӮпјҡ$("input[id][name$='man']")

еӯҗд»ЈиҝҮж»ӨеҷЁ Child Filters

  • :first-childйҖүжӢ©еҗҢзҲ¶д»Јзҡ„з¬¬дёҖдёӘеӯҗд»Је…ғзҙ гҖӮ
  • :first-of-typeйҖүжӢ©еҗҢдёҖе…ғзҙ еҗҚз§°зҡ„е…„ејҹдёӯзҡ„з¬¬дёҖдёӘе…ғзҙ гҖӮ
  • :last-childйҖүжӢ©еҗҢзҲ¶д»Јзҡ„жңҖеҗҺдёҖдёӘеӯҗд»Је…ғзҙ гҖӮ
  • :last-of-typeйҖүжӢ©еҗҢдёҖе…ғзҙ еҗҚз§°зҡ„е…„ејҹдёӯзҡ„жңҖеҗҺдёҖдёӘе…ғзҙ гҖӮ
  • :nth-child()йҖүжӢ©еҗҢзҲ¶д»Јзҡ„з¬¬nдёӘеӯҗд»Је…ғзҙ гҖӮ
  • :nth-last-child()йҖүжӢ©еҗҢзҲ¶д»Јзҡ„еҖ’ж•°з¬¬nдёӘеӯҗд»Је…ғзҙ гҖӮ
  • :nth-last-of-type()йҖүжӢ©еҗҢзҲ¶д»Јзҡ„еҖ’ж•°з¬¬nдёӘеӯҗд»Је…ғзҙ гҖӮ
  • :nth-of-type()йҖүжӢ©еҗҢзҲ¶д»Јзҡ„з¬¬nдёӘеӯҗд»Је…ғзҙ гҖӮ
  • :only-childйҖүжӢ©еҸӘжңүдёҖдёӘеӯҗд»Јзҡ„е…ғзҙ гҖӮ
  • :only-of-type()йҖүжӢ©жүҖжңүжІЎжңүеҗҢеҗҚе…ғзҙ зҡ„е…„ејҹе…ғзҙ гҖӮеҰӮпјҡ$("div.button:only-child")йҖүжӢ©еҸӘжңүдёҖдёӘbuttonзҡ„div
