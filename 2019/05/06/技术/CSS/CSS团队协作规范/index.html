  • class nameзҡ„е‘ҪеҗҚеҝ…йЎ»жҳҜиЎҢдёәгҖҒжңүиҜӯж„Ҹзҡ„гҖӮ

  • CSSжң¬иә«жңүжқғйҮҚи®ҫи®ЎпјҢд»»ж„Ҹең°дҪҝз”Ё!importantдјҡйҖ жҲҗжқғйҮҚж··д№ұиҖҢж— жі•з»ҙжҠӨгҖӮ

дёҚеҸҜиҪ»жҳ“йҷҗе®ҡе®Ҫй«ҳ

  • з”ЁжҲ·еҸҜд»ҘиҮӘиЎҢи®ҫе®ҡиҮӘе·ұзҡ„жөҸи§ҲеҷЁпјҢдҫӢеҰӮAndroidжүӢжңәеҸҜд»Ҙи®ҫе®ҡжҳҫзӨәеӯ—дҪ“еӨ§е°ҸпјҢеҶҷжӯ»зҡ„й«ҳеәҰдјҡи®©еӯ—дҪ“зӣёдә’йҮҚеҸ гҖӮ

RWDеӨұж•Ҳ

  • з§»еҠЁи®ҫеӨҮзҡ„й«ҳеәҰжҳҜж— йҷҗзҡ„пјҢе®ҪеәҰжҳҜжңүйҷҗзҡ„гҖӮ
  • иҜ·дёҚиҰҒжҠҠе®ҪеҶҷжӯ»гҖӮ

imgиҜ·и®©е®ғиҮӘеҠЁзј©ж”ҫ

  • иҜ·дёҚиҰҒжӣҝimgзҡ„е®№еҷЁи®ҫе®ҡе®ҪжҲ–й«ҳпјҢи®©е®ғж №жҚ®и®ҫеӨҮиҮӘиЎҢзј©ж”ҫгҖӮ
  • иҜ·дҪҝз”Ёbootstrapзҡ„img-responsiveгҖӮ
  • еҰӮжһңзЎ®е®һиҰҒз”Ёпјҡ
  • еҰӮжһңиҰҒз»ҷimg border-radiusи®ҫе®ҡж ·ејҸпјҢиҜ·дҪҝз”ЁзҲ¶е…ғзҙ жҺ§еҲ¶иЎҢдёәпјҢдҝқжҢҒimgеҸӘиҪҪе…ҘеӣҫзүҮпјҢжІЎжңүж ·ејҸгҖӮ

дјҳе…ҲдҪҝз”ЁgridжҺ’зүҲ

  • иҜ·дёҚиҰҒиҠұеҫҲеӨҡж—¶й—ҙеңЁеҶҷmedia queryпјҢи®ҫе®ҡдёҖе ҶbreakpointпјҢиҮӘе·ұеҶҷз»„д»¶ж ·ејҸпјҢиҮӘе·ұжҺ§еҲ¶жҜҸз§Қи®ҫеӨҮдёҠзҡ„е®№еҷЁе®ҪеәҰгҖӮ
  • иҜ·дҪҝз”Ёgrid systemпјҢиҝҷдәӣйғҪжҳҜе·Із»ҸжҲҗзҶҹзҡ„жЎҶжһ¶пјҢиҖҢдё”жңүдәӣд№ҹе·Із»Ҹеё®дҪ еӨ„зҗҶдәҶжөҸи§ҲеҷЁзӣёе®№й—®йўҳгҖӮ

дёҚеҸҜзӣҙжҺҘover writeжҲ–еңЁеҺҹжң¬жЎҶжһ¶зҡ„classеўһеҠ еҶ…е®№

  • зӣҙжҺҘеҶҷдёҖдёӘж–°зҡ„classпјҢдёҚиҰҒиҰҶзӣ–еҺҹжңүзҡ„и®ҫи®ЎгҖӮ
  • дёҚиҰҒеҶҚе·Іжңүзҡ„classдёҠж·»еҠ ж ·ејҸпјҢиҜ·йўқеӨ–еҚ•зӢ¬еҶҷдёҖдёӘгҖӮ

и®ҫи®ЎRWDзҪ‘з«ҷпјҢиҜ·йҒөеҫӘз§»еҠЁи®ҫеӨҮдјҳе…ҲеҺҹеҲҷ

  • и®ҫи®ЎеёҲзҡ„и®ҫи®ЎйЎәеәҸпјҢд»ҘжЎҢйқўзүҲдјҳе…ҲпјҢеҶҚи®ҫи®ЎжүӢжңәзүҲгҖӮ
  • еүҚз«Ҝе·ҘзЁӢеёҲжӢҝеҲ°и§Ҷи§үеӣҫпјҢејҖе§ӢеҶҷHTML/CSSж—¶д»ҘжүӢжңәзүҲдёәз¬¬дёҖдјҳе…ҲгҖӮ

жүӢжңәејҖеҗҜзҪ‘йЎөеҫҲеҗғжүӢжңәж•ҲиғҪе’ҢзҪ‘з»ңзҠ¶еҶөпјҢеүҚз«Ҝе·ҘзЁӢеёҲдёҖејҖе§Ӣе°ұд»ҘжүӢжңәзүҲдёәдјҳе…ҲпјҢеҸҜд»Ҙи®©HTMLдёҖејҖе§ӢиҪҪе…ҘпјҢдҪҝз”ЁжңҖе°‘зҡ„ж•ҲиғҪеҝ«йҖҹиҪҪе…ҘзҪ‘йЎөгҖӮеҪ“ејҖе§ӢеҲ¶дҪңжЎҢйқўзүҲж—¶пјҢеҸӘдјҡе°‘и®ёи·‘зүҲпјҢеҒҡйҖӮеҪ“еҫ®и°ғеҚіеҸҜгҖӮ
зӣёеҸҚпјҢеҰӮжһңе…ҲеҲ¶дҪңжЎҢйқўзүҲпјҢеҪ“жүӢжңәзүҲз”»йқўиў«еҲҮжҺүжҲ–жҳҜи·‘зүҲпјҢйңҖиҰҒиҠұжӣҙеӨҡж—¶й—ҙеҺ»и°ғж•ҙгҖӮ

еҶҚжқҘжҳҜiPhoneжүӢжңәзҡ„retinaпјҢдјҡе°ҶеӣҫзүҮж”ҫеҲ°жүӢжңәдёҠж—¶иҮӘеҠЁеҒҡдёӨеҖҚзј©е°ҸпјҢеңЁдёҖејҖе§ӢеҲ¶дҪңж—¶еҚіеҸҜеҸ‘зҺ°еӣҫзүҮиҪҪе…ҘжҳҜеҗҰеҗғж•ҲиғҪгҖӮдёәдәҶи®©еӣҫзүҮиғҪеңЁiPhoneдёҠжңүжӣҙеҘҪзҡ„дҪ“йӘҢпјҢе»әи®®з§»еҠЁзүҲдјҳе…ҲгҖӮ
