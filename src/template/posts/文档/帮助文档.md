### `Prettier` 注释指令对 `Markdown` 选定内容不格式化

1. **忽略单个代码块**
   在代码块前一行添加 `<!-- prettier-ignore -->` 注释

```javascript
<!-- prettier-ignore -->
// 这段代码会保持原样，不会被Prettier格式化
const   messyCode   =   {   a: 1,   b: 2   };
```

2. **忽略多行/整个区域**
   使用 `<!-- prettier-ignore-start -->` 和 `<!-- prettier-ignore-end -->` 包裹需要忽略的内容

```javascript
<!-- prettier-ignore-start -->
这部分内容
    会完全保持原样
        包括缩进和换行
都不会被Prettier修改

# 这个代码块也会被忽略
print(   "hello"   )
<!-- prettier-ignore-end -->
```

3. **忽略整个 `Markdown` 文件**
   在项目根目录创建 `.prettierignore` 文件，并添加 `*.md`

   **忽略代码块内的特定语言**

   只想禁用代码块内某一种语言的格式化，保留其他语言的格式化，可以在 `.prettierrc` 配置文件中添加：

```json
{
  "overrides": [
    {
      "files": "*.md",
      "options": {
        "embeddedLanguageFormatting": "off"
      }
    }
  ]
}
```
