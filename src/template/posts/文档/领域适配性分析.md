## 领域分析

1. **前端 `AI` 开发**

- 音视频（`WebRTC`、流媒体）、数据可视化 `Echarts`、`Node.js` 后端服务
- 智能行为分析、`AI`身份核验、`AI`运维
- `AI`能力前端集成 + `AI`硬件对接 + `AI`交付
- 基础：`TensorFlow.js`轻量化模型部署、主流大模型 `API` 调用
- 核心：`AI` 音视频处理（降噪/增强/行为识别）、`AI` 硬件 `SDK` 封装、`AI` 分析结果可视化
- 差异化：国产化 `AI` 框架对接（华为昇腾/百度飞桨）、离线 `AI` 推理

1. **`Node.js + Python` 全栈开发**

- 全栈能力：前端（`React/Vue`） + `Node.js`(实时服务、信令交换、文件传输) + `Python`（数据/接口/自动化）
- `Node.js`：深化 `NestJS`（企业级框架）、`PM2` 部署、日志/监控
- `Python`：`FastAPI`（接口）+ `Pandas` （数据处理）

3. **`React Native`**

- `React Native`的核心逻辑（跨端渲染、原生桥接），状态管理（`Redux/Zustand`）, `RN` 生态（组件库、原生模块封装）
- `RN` 核心：组件库（`React Native Paper/Ant Design Mobile`）、状态管理、原生模块（`Android`）对接硬件 `SDK`
- `AI` 移动端集成：`TensorFlow Lite` 、移动端 `AI SDK` 对接

4. **`GIS` 开发**

- 掌握 `ECharts` 数据可视化，`GIS` 前端核心就是 **前端 + 地图 `SDK` + 数据可视化**
- `GIS` 地图 + `AI` 数据可视化
- `AI` + 安防 `GIS`（摄像头点位监控、异常行为热力图）
- `AI` + 电网 `GIS` （设备巡检、故障预警）
- 基础：坐标系、`GeoJSON`、`OpenLayers/ArcGIS JS API`
- 进阶：`GIS` + `AI` 可视化、空间数据分析
- 实战：基于 `Openlayers` 做 `AI` 安防 `GIS` 系统，实现摄像头点位标注、异常行为热力图、实时预警推送

5. **`Web3 DApp` 开发**

- `React + Ethers.js` 做 `DApp` 开发
