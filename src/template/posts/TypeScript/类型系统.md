- 类型安全：强类型 弱类型
- 类型检查：静态类型 动态类型
- 标准库：内置对象对应的声明

## 类型

- 类型注解 `let a:string = 'hello'` 类型推断
- 联合类型 `let a:string | number = 'hello'` 类型之间进行 **或** 操作
- 交叉类型 类型之间进行 **与** 操作
- `never`类型 永不存在值的类型
- `any`类型 任意类型 不进行检测
- `unknown`类型 `any`类型对应的安全类型 默认会进行检测
- 类型断言 `a as string`
- 非空断言 `a!.length`
- 数组类型
  - 类型 `[]`：`let a:(number|string)[] = [1,2,3]`
  - `Array`类型：`let a:Array<number> = [1,2,3]`
- 元祖类型 `let arr:[number,string]=[1,'hello']`
- 对象类型 `type A = {name: string, age?: number}`
- 索引签名 `type A = {name: string, age?: number, [index:string]: any}`
- 函数重载和可调用注解
  ```typescript
  type A = {
    (n: number): number;
    m?: string;
  };
  ```
- 枚举类型和 `const`枚举 `const`编译后的优化
  ```typescript
  enum Roles {
    SUPER_ADMIN,
    ADMIN = 3,
    USER,
  }
  enum Roles1 {
    SUPER_ADMIN = 'super_admin',
    ADMIN = 'admin',
    USER = 'user',
  }
  const enum Roles2 {
    SUPER_ADMIN = 'super_admin',
    ADMIN = 'admin',
    USER = 'user',
  }
  ```
