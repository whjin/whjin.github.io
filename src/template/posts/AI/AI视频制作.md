# 完整制作全流程

核心逻辑：豆包负责全案策划、脚本分镜、提示词生成与优化，即梦AI负责角色定妆、画面生成、动态视频制作

二者联动解决漫剧创作中“剧情乱、人物崩、画面散、动效差”四大核心痛点

## 一、前期策划与剧本创作（豆包，5分钟）

核心目标：题材、人设、完整剧情，生成适配即梦AI的合规剧本，规避审核风险

给豆包发送精准指令，直接生成可落地剧本：

1. 核心操作

> 你是专业漫剧编剧，帮我创作一个【时长，如1分钟】的【题材，如都市甜宠/古风仙侠/悬疑反转/校园治愈】竖屏漫剧剧本，适配抖音/视频号9:16比例，用即梦AI Seedance2.0模型生成，全程规避违规内容。要求：1、先输出核心人设（2-3个核心角色，标注固定外貌、服饰、性格特征，方便后续锁定人物）；2、剧情节奏紧凑，每3-5秒一个剧情节点，开头抓眼球，结尾留悬念；3、标注每段台词、对应情绪和推荐音效。

2. 关键要点
   - 人设必须加固定识别特征（如齐刘海+银框眼镜、红色发带、左眼泪痣），是后续人物不崩脸的核心
   - 剧情避免大段对白，单句台词不超过20字，适配短视频节奏
   - 提前告知豆包适配即梦模型，规避敏感内容，减少后续生成失败概率

## 二、标准化分镜脚本生成（豆包，3分钟）

核心目标：把剧本拆解成AI可识别的分镜镜头，每个镜头自带生图/生视频提示词

1. 生成标准化分镜

   > 把剧本拆成【6-8个】竖屏9:16漫剧分镜，每个分镜必须标注：镜号、时长、景别、画面描述、台词、音效、即梦生图提示词，全程保持人物设定统一，镜头语言符合漫剧节奏。

2. 分镜黄金法则采用三镜组合法
   - 全景（2-3秒）：交代场景环境、人物位置，定整体氛围
   - 中景（3-4秒）：展现人物动作、互动关系，推进剧情
   - 特写（2-3秒）：捕捉人物表情、关键细节，放大情绪

## 三、角色定妆与风格锁定（即梦、豆包，10分钟）

核心目标：生成角色定妆照，锁定人物五官、服饰、画风

1. 角色定妆照
   - 豆包优化定妆提示词：把人设发给豆包，指令“把这个人设优化为即梦AI文生图专用的角色定妆提示词，补充画质、光影、风格细节”
   - 即梦生成定妆照：选择「文生图」，比例选9:16（竖屏漫剧首选），粘贴提示词，生成角色定妆照
   - 关键设置：每个角色生成2-4张不同角度（正面/侧面/45度）的定妆照，保存备用，后续所有镜头生成上传对应定妆照作为参考

## 四、分镜画面批量生成（即梦，豆包，20分钟）

核心目标：按分镜脚本，生成风格统一、人物一致的高清分镜图

1. 即梦「文生图」，上传对应角色的定妆图，开启「角色锁定/参考图模式」
2. 对应分镜提示词，保持参数和定妆照一致，逐镜生成画面
3. 对不满意的单图，用「局部重绘」微调，不改动整体画风和人物特征
4. 多角色同框镜头，在提示词里用`@图片1``@图片2`标记对应角色定妆照，AI精准匹配人物特征，避免同框变脸

## 五、动态漫剧/动画片段是生成（即梦，豆包，30分钟）

核心目标：把静态分镜图转为动态视频，添加运镜、动作、音效，实现漫剧动画效果。

**方式1：图生视频**

适合静态分镜转动态，精准控制画面

1. 即梦「图生视频」，上传单张分镜图
2. 动态提示词，明确动效、运镜、音效要求
3. 设置时长（免费额度推荐3-10秒），模型Seedance2.0
4. 用「动态笔刷」精准控制运动区域（只让头发飘动、手指动），「运镜控制」选择推/拉/摇/移，实现电影感镜头
5. 逐镜生成所有分镜的动态片段，导出备用

**方式二：文生视频**

适合连贯剧情片段，直接生成代配音

1. 即梦「视频生成」，模型切换为Seedance2.0，选择「全能参考」模式
2. 上传所有角色定妆照，在提示词里用 `@图片` 标记对应角色
3. 分镜全案提示词，明确时长、画面、动作、运镜、台词
4. 设置视频比例9:16，时长按需选择，开启「音画同步」，AI自动生成对口型的配音和匹配音效
5. 生成后预览，用「局部优化」微调动效和画面细节

## 六、后期合成与成片导出（5分钟）

1. 把所有动态片段导入剪映等剪辑工具，按分镜顺序拼接
2. 添加字幕、片头片尾、背景音乐、微调音量，匹配台词和画面节奏
3. 检查画面连贯性，导出1080高清成片，漫剧/动画制作完成

# 避坑核心要点

1. **人物一致性是核心**：全程用同一套定妆照作为参考，开启角色锁定，提示词里固定人物核心特征，中途不修改画风和五官描述
2. **风格全程统一**：一旦选定国漫 / 韩漫 / 写实 / 二次元风格，所有镜头的风格关键词必须完全一致，避免出现画风撕裂；
3. **免费额度最大化**：即梦新用户注册送免费生成次数，每日登录可领取额外额度，新手优先用图生视频，消耗额度更少，稳定性更高；
4. **审核规避**：避免生成暴力、血腥、低俗、敏感题材，剧本提前让豆包做合规优化，减少生成失败；
5. **动效适度原则**：单镜头只保留 1-2 个核心动态，避免全画面乱动，导致画面穿模、人物变形。
